Cloudflare Setup Guide

Follow these steps to connect your Cloudflare Worker backend. Takes about 5 minutes.

1
Create a KV Namespace
Go to workers.cloudflare.com → KV (left menu) → Create namespace
Name it SV_KV and click Add.
2
Create a Worker
Go to Workers & Pages → Create → Create Worker
Name it streamvault, click Deploy, then click Edit Code.
3
Paste worker.js Code
Delete all existing code. Paste the entire contents of worker.js you downloaded. Change the password at the top if you want, then click Save & Deploy.
4
Bind KV to Worker
Go to your Worker → Settings → Variables → KV Namespace Bindings
Click Add binding:
Variable name: SV_KV
KV namespace: select your SV_KV namespace
Click Save.
5
Copy your Worker URL
On your Worker page, your URL will look like:
https://streamvault.yourname.workers.dev
Copy this URL.
6
Update streamvault-netflix.html
Open streamvault-netflix.html and find:
YOUR_WORKER_URL
Replace it with your actual Worker URL. Save the file.
7
Log into this Admin
Enter your Worker URL and password in the login screen. All your content is now stored in Cloudflare KV — changes are live on your site instantly. No more manual exports!
Free Tier Limits: 100,000 Worker requests/day, 1GB KV storage, unlimited reads. More than enough for a streaming site.
